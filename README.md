# Task Manager

## Project Overview
This project is a Task Manager application designed to help users efficiently manage their daily tasks. It allows users to create, update, and delete tasks, as well as track their progress.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/LauraCodeBrave/Task_manager.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Task_manager
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and access the application at `http://localhost:3000`.
